사전준비사항

정보 보기

고객님께서 인터넷을 통하여 주택을 신청하기 위해서는 다음과 같은 사항들을 사전에 준비하셔서 입력된 자료가 착오 또는 사실과 다르게 신청되어 불이익을 받는 일이 없도록 해주시기 바랍니다.(개인용 공동인증서 또는 간편인증 또은 금융인증서를 인터넷 청약신청 접수일 이전에 미리 발급 완료)

준비사항, 비고 항목으로 구성된 사전준비사항 표
준비사항 비고
공동인증서, 금융인증서, 간편인증 사전발급 공사는 본인 확인에 대한 절차로서 공동인증서, 금융인증서, 간편인증을 사용합니다.

※ 아래 공동인증서 발급안내 참고
도로명주소 숙지

새로운 주소(도로명 주소)로만 청약을 받습니다.

도로명주소 안내시스템 바로가기
참고사항

※ 공동인증서가 안되시는 분께서는 간편인증 혹은 금융인증서를 이용하시기 바랍니다.

※ 간편인증 로그인 후 실명확인이 안되시는 분께서는 공동인증서 혹은 금융인증서를 이용하시기 바랍니다.

공동인증서 발급 안내

정보 보기

은행에서 발급

  • 인터넷 뱅킹 신청 : 가까운 은행에서 인터넷 뱅킹을 신청하셔서 신원확인 절차를 하시면 됩니다.
  • 금융거래인증서 발급 : 인터넷 뱅킹을 신청한 은행의 홈페이지에서 공동인증서를 발급 받으시면 됩니다.

인증기관에서 발급(유료)

  • 공동인증서는 반드시 개인용 공동인증서여야 합니다.
  • 만료일이 지난 인증서는 사용이 불가하니, 반드시 만료일을 확인하시기 바랍니다.

인증서 창이 뜨지 않을 경우 '수동설치 프로그램' 설치하기

  1. 1. 시작-제어판-프로그램 및 기능에서 [VestCert] 삭제
  2. 2. 아래 수동설치 프로그램을 설치 후 이 창을 닫으신 후 F5키를 눌러 새로고침을 하시기 바랍니다.

FAQ(모바일)

네이버앱, 다음앱 이용 시 청약신청이 잘 안 되는 경우

정보 보기
  • 모바일 청약시 네이버앱/다음앱 같은 경우는 청약시 시스템과 호환성 문재로 인하여 청약이 정상적으로 진행이 되지 않습니다.
    안드로이드 휴대폰은 내부의 인터넷 브라우저 또는 크롬 브라우저를 사용 하시기 바랍니다
    아이폰은 크롬 또는 사파리 브라우저를 이용하시기 바랍니다.

다음페이지 이동시 로그인 정보가 없습니다 경고창이 뜬 경우

정보 보기
  • 해당 브라우저 닫기, 혹은 어플리케이션 모두 닫기 후 새로 인터넷 브라우저 창을 열어서 다시 접속하여 청약 신청을 합니다.

FAQ(PC)

공동인증서가 안보이는경우

정보 보기
  • 공동인증서가 PC혹은 휴대폰에 있는지부터 확인 후 없다면 해당은행으로 연락 바랍니다.
  • 개인인증서만 지원가능(내국인, 외국인 선택 시 개인인증서만 보임, 사업자 선택 시 사업자 인증서만 보임)(사업자 인증서는 용지청약, 선착순 분양만 가능하고 나머지는 사업자 인증서 지원 안됨)

도로명주소가 검색 안되는 경우

정보 보기

검색방법

  • 동/리와 지번을 검색 하세요. 예) 개포동 14-5
  • 건물명으로 검색 하세요. 예) SH공사
  • 도로명으로 검색 하세요. 예) 개포로 621
  • 동명은 붙여서 써 주세요. 예) 개포1동(○), 개포 1동(X)
  • 동명 검색 시 '제'는 빼고 입력하여 주세요. 예) 개포1동(○), 개포제1동(X)
  • 찾으시는 주소가 없으신 경우 1600-3456 번으로 연락하시기 바랍니다.

다음페이지 이동시 로그인 정보가 없습니다 경고창이 뜬 경우

정보 보기
  • 메세지 출력시 열려 있는 인터넷 브라우저 창을 모두(현재 청약 창도 닫음) 닫은 후(모바일인 경우에는 해당 브라우저 닫기, 혹은 어플리케이션 모두 닫기) 새로 인터넷 브라우저 창을 열어서 다시 접속하여 청약 신청을 합니다.

IE 브라우저에서 체크 박스 선택이 안되는 경우

정보 보기
  • 사용자별 개인 PC에 설치된 OS 브라우저 버젼와 인터넷 익스플러어(IE) 버젼과 맞지 않아서 라디오 버튼이 체크 박스로 보이면서 체크가 안되는 현상이 발생하기도 합니다.
    이럴때는 현재 설치된 인터넷 익스플로어(IE)를 종료 하시고, Chrome(크롬)브라우저, 엣지 브라우저를 사용하시면 정상적으로 체크 선택이 가능합니다.

최종 확인 시 네트워크 오류가 난 경우

정보 보기
  • 초/중/고등학교 혹은 일부 회사의 네트워크는 보안으로 막혀있기 때문에 최종공동인증이 되지 않는 경우가 있습니다.
    네트워크오류(예: “네트워크 오류입니다.”)가 뜬다면 해당위치의 컴퓨터에서는 청약을 진행할 수 없으므로 모바일 혹은 다른장소의 PC로 진행하여야 합니다.

키보드 보안 프로그램 관련 설명

정보 보기

'사이트의 프로그램'들이 설치가 되지 않으신다면

  • "백신프로그램으로" 고객님 pc의 바이러스 체크 및 사이트의 프로그램들이 설치가 되지 않으신다면, 애드웨어 설치여부를 확인하여 주십시요!
  • (바이러스 또는 애드웨어가 설치되어 있는 경우는 프로그램들이 정상적으로 설치되지 않습니다.)
    "내부 보안망이 설치된 사무실" 에서 청약을 신청하시는 경우는 사내 네트워크 담당자에게 문의하여 해결하시기 바랍니다.

정보 보기

도구 > 인터넷옵션 > 일반 탭 > 탭 버튼을 누르고 아래와 같이 설정하시기 바랍니다 .

스마트폰으로 인증서 내보내기 이용절차

스마트폰으로 인증서 내보내기 이용

  • 스마트폰 공동인증서는 PC의 공동인증서를 단순 복사하는 것으로, PC의 공동인증서에는 아무런 영향을 주지 않습니다.
  • 스마트폰으로 복사 된 인증서는 해당 모바일기기에서만 사용이 가능합니다.
    (모바일 인증서를 다시 PC로 이동 불가, 모바일 인증서로 PC 홈페이지 접속 불가)
  • 공동인증서 만료일이 지나면 PC의 공동인증서를 재발급 받은 후 해당 인증서를 다시 복사해야 합니다.
    ※ '인증서 내보내기' 클릭 후 시간이 소요될 수 있습니다.

Step1

스마트폰으로 인증서 보내기 버튼 클릭

  • PC에서 인터넷청약 홈페이지(http://www.i-sh.co.kr/app/index.do) 접속
  • 메인 화면 상단의 공동인증서 로그인 메뉴 클릭
  • '스마트폰으로 인증서 내보내기' 버튼 클릭

Step2

인증서 선택화면

  • PC에서 스마트폰으로 복사할 인증서 선택
  • 인증서 비밀번호 입력

Step3

핸드폰에서 전자서명 가져오기

  • Mobile에서 인터넷청약 홈페이지(http://www.i-sh.co.kr/app/index.do) 접속
  • 본인의 주민등록번호 13자리를 입력 후 [공동인증서 로그인]버튼을 터치

Step4

승인번호 확인

  • [스마트폰] 인증번호 12자리 확인
  • [PC] PC에서 내보낼 공동인증서 암호와 인증번호 12자리 입력확인 클릭
  • [PC] 내보내기 클릭

Step5

핸드폰으로 가져온 인증서 확인

  • [스마트폰] 확인버튼 선택
  • [스마트폰] 청약 신청서 작성
